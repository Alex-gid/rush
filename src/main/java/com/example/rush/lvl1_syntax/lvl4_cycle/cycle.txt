Циклы — это конструкции в программировании, которые позволяют выполнять блок кода несколько раз.
Они используются для автоматизации повторяющихся задач.

Основные типы циклов в Java:
------------------------------------------------------------------------------------------------------------------------
for:
Используется, когда известно количество итераций.
Синтаксис:
for (инициализация; условие; обновление) {
    // тело цикла
}
Пример:
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
------------------------------------------------------------------------------------------------------------------------
while :
Используется, когда условие проверяется перед каждой итерацией.
Синтаксис:
while (условие) {
    // тело цикла
}
Пример:
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
------------------------------------------------------------------------------------------------------------------------
do-while :
Используется, когда нужно выполнить тело цикла хотя бы один раз, а затем проверять условие.
Синтаксис:
do {
    // тело цикла
} while (условие);
Пример:
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
------------------------------------------------------------------------------------------------------------------------
Вложенные циклы:
Циклы могут быть вложенными, то есть один цикл может содержаться внутри другого.
Пример:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        System.out.println("i = " + i + ", j = " + j);
    }
}
------------------------------------------------------------------------------------------------------------------------
Советы:
Избегайте бесконечных циклов: Убедитесь, что условие цикла рано или поздно станет ложным.
Оптимизация: Старайтесь минимизировать количество итераций, особенно во вложенных циклах.
Отладка: Используйте System.out.println() для вывода промежуточных значений переменных, чтобы понять, как работает цикл.
